# linuxdo-beautification-plugin

用于 LINUX DO（https://linux.do）的 Tampermonkey 美化脚本。

## 当前功能

- Apple 风格界面：统一字体、材质、导航、主题列表与帖子阅读流。
- 书签颜色标识：对可识别的书签主题、书签帖子和书签按钮增加醒目的颜色/边框标识。
- 关注作者高亮：每天首次进入网站时同步一次当前账号的全部关注用户并缓存到本地，新关注或取消关注时增量更新名单。
- 关注作者徽标：在主题列表头像和帖子作者用户名旁显示“已关注”。
- 关注通知样式：区分关注用户新主题、新回复以及新增关注者三类官方通知。
- 关键词高亮：在脚本菜单中配置关键词和对应颜色，仅在命中主题标题时高亮显示。
- 代码块复制按钮：帖子内的代码块右上角显示 Apple 风格胶囊复制按钮，支持一键复制。
- 长图自动折叠：超过 520px 的图片默认折叠，点击展开或收起。
- 引用折叠：多层嵌套或超长引用默认收起，点击标题栏可展开。
- 主题状态徽章：主题列表标题旁用彩色胶囊突出显示已解决、置顶、精华、抽奖、热门、已关闭等状态。
- 已读主题淡化：已读过的主题标题淡化显示，突出未读内容，悬停时恢复。
- 悬浮快捷操作：右下角显示"回到顶部/回到底部"按钮，向下滚动一定距离后出现。
- 功能开关面板：所有增强特性可在设置面板中单独启用或禁用。

## 使用方式

1. 在 Tampermonkey 中新建脚本。
2. 使用 [`linuxdo-beautification.user.js`](./linuxdo-beautification.user.js) 的内容。
3. 保存后访问 `https://linux.do`。
4. 在 Tampermonkey 菜单中打开“LINUX DO 美化设置”，配置关注高亮颜色和关键词规则。

## 实现说明

- 关注名单每天最多请求一次官方 `/u/{username}/follow/following.json` 接口；按账号和日期缓存，并使用跨标签页锁避免并发重复请求。
- 主题和帖子增强不会额外请求接口；刷新列表时只读取本地关注名单和页面已有数据。
- LINUX DO 是动态渲染页面，脚本通过 `MutationObserver` 和路由监听在页面更新后重新应用效果。
- 帖子保持 Discourse 原生顺序和回复交互，脚本不再移动帖子节点。
- 高亮配置使用 Tampermonkey 存储保存，关键词匹配不区分大小写。

## 后续可优化清单

- 夜间模式微调：为深色主题单独优化书签标识颜色。
