# linuxdo-beautification-plugin

用于 LINUX DO（https://linux.do）的 Tampermonkey 美化脚本。

## 当前功能

- 帖子回复楼中楼：在主题页读取 Discourse 的 `reply_to_post_number`，把“回复某楼”的帖子移动到父楼层下方展示。
- 书签颜色标识：对可识别的书签主题、书签帖子和书签按钮增加醒目的颜色/边框标识。
- 关注作者高亮：自动读取站点暴露的关注状态，关注用户发帖时高亮显示。
- 关键词高亮：在脚本菜单中配置关键词和对应颜色，命中主题标题或帖子内容时高亮显示。

## 使用方式

1. 在 Tampermonkey 中新建脚本。
2. 使用 [`linuxdo-beautification.user.js`](./linuxdo-beautification.user.js) 的内容。
3. 保存后访问 `https://linux.do`。
4. 在 Tampermonkey 菜单中打开“LINUX DO 美化设置”，配置关注高亮颜色和关键词规则。

## 实现说明

- 脚本优先读取页面内 Discourse 模型中的帖子数据；读不到时回退到当前主题的 `.json` 数据。
- LINUX DO 是动态渲染页面，脚本通过 `MutationObserver` 和路由监听在页面更新后重新应用效果。
- 楼中楼重排是幂等处理：已在正确父楼下的帖子不会重复移动，避免鼠标悬停或页面局部更新时闪烁。
- 高亮配置使用 Tampermonkey 存储保存，关键词匹配不区分大小写。

## 后续可优化清单

- 主题列表增强：高亮热门、已解决、精华、抽奖或高级推广主题。
- 阅读体验：长帖目录、只看楼主、折叠长引用、代码块复制按钮。
- 图片体验：图片最大宽度、预览增强、长图折叠。
- 交互效率：快捷返回顶部/底部、快速跳转指定楼层、自动展开被折叠回复。
- 个性化设置：提供开关面板，可单独启用/禁用每个功能。
- 夜间模式微调：为深色主题单独优化楼中楼和书签标识颜色。
